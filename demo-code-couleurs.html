§ Exemples de codes : l'information donnée uniquement par la couleur

§ Information donnée uniquement par la couleur dans un menu

Mauvaise pratique

<ul class="menu">
    <li><a href="#">Actualités</a></li>
    <li class="active"><a href="#">Le ministère</a></li>
    <li><a href="#">Vos questions</a></li>
    <li><a href="#">Contactez nous</a></li>
</ul>
  • Actualités
  • Le ministère
  • Vos questions
  • Contactez nous

Bonnes pratiques avec HTML

<ul class="menu">
    <li><a href="#">Actualités</a></li>
    <li class="active"><a href="#" style="border: 1px solid black" title="Le ministère, rubrique active">Le ministère</a></li>
    <li><a href="#">Vos questions</a></li>
    <li><a href="#">Contactez nous</a></li>
</ul>
  • Actualités
  • Le ministère
  • Vos questions
  • Contactez nous

Bonnes pratiques avec ARIA

<ul class="menu">
    <li><a href="#">Actualités</a></li>
    <li class="active"><a href="#" style="border: 1px solid black" aria-current="page">Le ministère</a></li>
    <li><a href="#">Vos questions</a></li>
    <li><a href="#">Contactez nous</a></li>
</ul>
  • Actualités
  • Le ministère
  • Vos questions
  • Contactez nous
