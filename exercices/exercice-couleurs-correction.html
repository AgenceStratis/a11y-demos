Correction de l'exercice : les couleurs

Information donnée uniquement par la couleur dans un menu

  • Actualités
  • Le ministère
  • Vos questions
  • Contactez nous

Autres possibilités :

  • Remplacer l'attribut title par un attribut aria-label.
  • Utiliser l'attribut aria-current="page" sur le lien.

À noter

Ces solutions fournissent l'information à tous les utilisateurs concernés :

  • La bordure, pour les utilisateurs mal-voyants qui ne voient pas correctement les couleurs.
  • Les attributs title, aria-label ou aria-current, pour les utilisateurs non-voyants qui ne peuvent pas "voir" la page.

Contraste de couleurs sur un élément textuel

Autres possibilités :

  • Remplacer (éclaircir) les couleurs de fond.
  • Sur le bouton 1, augmenter la taille du texte à 2.4rem.
  • Sur le bouton 1, passer le texte en gras.

Rappel

Texte non gras :

  • < 24px : ratio 4.5:1 entre la couleur de fond et le texte
  • >= 24px : ratio 3:1 entre la couleur de fond et le texte

Texte gras :

  • < 18.5px : ratio 4.5:1 entre la couleur de fond et le texte
  • >= 18.5px : ratio 3:1 entre la couleur de fond et le texte

Vérifier aussi les états survolés, focussés, actifs...

Contraste de couleurs sur un élément non textuel

Rappel

Ratio 3:1 entre la couleur de fond et celle du composant, ou entre toutes les couleurs contiguës d'un graphique

Vérifier aussi les états survolés, focussés, actifs...
