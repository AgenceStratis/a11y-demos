Démonstrations en accessibilité

Images

$ Légende d'image avec le couple <figure>, <figcaption>

Code

<figure role="figure" aria-label="Copyright 2016 auteur">
    <img alt="alternative de l’image" src="https://picsum.photos/200/300"/>
    <figcaption>Copyright 2016 auteur</figcaption>
</figure>

$ Légende d'image avec le role figure et l'attribut aria-labelledby

Cette implémentation n'est pas conforme à l'accessibilité car le support dans les agents utilisateurs n'est pas suffisant.

Code

<div role="figure" aria-labelledby="Copyright 2016 auteur">
  <img src="image.png" alt="alternative de l’image">
  <p id="caption">Copyright 2016 auteur </p>
</div>
